## 关于我

我主要关注企业级中后台、业务系统、权限模型、数据建模与复杂业务流程的工程化实现。

日常工作更偏向把真实业务中的复杂流程，沉淀成清晰、稳定、可维护的系统能力：后端服务、管理后台、数据结构、权限体系、菜单角色、流程状态，以及前后端之间可靠的协作边界。

我相信好的系统不只是在当前需求下能跑通，还应该能支撑下一次需求变更、下一次团队协作，以及下一位维护者的理解成本。
