## Profile

I build software around real operational work: backend services, admin platforms, data models, permission systems, workflow states, and the tools that help teams run complicated business processes with less friction.

My preferred kind of system is clear on the surface and strict underneath: understandable UI, stable APIs, explicit data rules, traceable permissions, and changes that can be verified.

## What I Build

| Area | Focus |
| --- | --- |
| Business Systems | Order flows, resource management, permissions, menus, roles, and operational dashboards |
| Backend Engineering | Java, Spring Boot, modular services, transactional correctness, API contracts |
| Admin Frontend | Vue 3, TypeScript, dense management pages, clear feedback, maintainable components |
